# Reto 3 - Krowdy Academy

Tercer reto de Krowdy Academy, realizado con React y Vite

# Como ejecutar

1.- Descargar este repositorio

2.- Abrir en un editor de código como VS Code

3.- Abrir un terminal y escribir 'npm install'. Esto instalará todas las dependencias listadas en package.json

4.- Una vez se instalen, escribir 'npm run dev' en el terminal. Esto iniciará el servidor Vite.

5.- Abrir el enlace listado en Vite para poder utilizar el programa
